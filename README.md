Official Python SDK for the [Food Context Protocol](https://github.com/Food-Context-Protocol/fcp).

## Installation

```bash
pip install fcp
```

## Quick Start

```python
from fcp import FCPClient

# Initialize client
client = FCPClient(
    base_url="https://fcp.dev",
    api_key="your-api-key"
)

# Analyze a meal photo
result = client.analyze.analyze_meal(
    image_url="https://example.com/food.jpg"
)
print(result.nutrition)

# Search recipes
recipes = client.recipes.search_recipes(
    query="high protein breakfast",
    max_results=5
)

# Check food recalls
recalls = client.safety.check_food_recalls(
    food_item="romaine lettuce"
)
```
